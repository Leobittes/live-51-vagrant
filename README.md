
O que é Vagrant?
Criar maquinas virtuais atraves de insfraestrutra como codigo.

O Vagrant so é possivel instalar na maquina fisica.

Vagrantfile
- SO = Boxe
- Memoria
- CPU
- DISCO
- REDE
- SINCRONIZAR UMA PASTA
- NOME DA MAQUINA
- PROVISION
    - SCRIPT
    - ANSIBLE
    - PUPPET

Comando na maquina fisica.    

vagrant init ubuntu/focal64
vagrant up
vagrant ssh
vagrant destroy
